# O Projeto

## Introdução

Este projeto esta sendo desenvolvido para o **Projeto de Inovação** do curso **Técnico em Edificações**, pela instituição _SESI SENAI CFP Celso Charuri_, em Pará de Minas.

## Tecnologias usadas

- Typescript
- React
- React-Native
- Expo
- Native-Base
- Phosphor Icons
